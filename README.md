_A lightweight CLI tool for listening to MAVLink messages over various connection types._

## **Installation**

### **1. Install via Cargo **

```sh
cargo install --path .
```

Or

```sh
cargo install mavshark
```

## **Usage**

### **Basic Command**

```sh
mavshark listen <ADDRESS> [OPTIONS]
```

```sh
mavshark help
```

```sh
mavshark listen --help
```
